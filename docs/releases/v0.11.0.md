# 发布说明：v0.11.0

## 摘要

- 收拢并稳定了 `scene -> chapter -> volume -> book` 的完整最小主链，形成可实际演示与交付的预览版基线。
- 完成成品化收尾，覆盖 `export / rebuild / diagnostics / health`、`project package / import-package / archive / backup / snapshots`。
- 增加发布前硬化资产：可直接导入的示例项目包、演示流程、发布检查清单、发布说明模板、只读 `release_preflight` 检查脚本。

- Scene Draft 最小生成链、deterministic checks、accept / reject、targeted repair。

- Chapter / Volume / Book assembled / finalized 主链。
- Book continuity checks 与 finalize preflight gating。

## 演示流程

推荐演示路径与 `examples/sample-project-package/` 强绑定：

1. 启动服务：`python -m uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000`
2. 导入示例项目包：`POST /api/projects/import-package`
3. 打开 `/studio`
4. 在工作台加载导入项目的体检信息
5. 触发：
   - `导出项目包`
   - `创建归档快照`
   - `创建备份`
   - `加载快照列表`

详细步骤与每一步预期结果见：

- `docs/demo-workflow.md`

## 已知限制

- 当前版本仍是预览版，不是最终正式发布。
- 不新增新的智能写作模块。
- 不做 LLM critic。
- 不做 book-level repair / rewrite / polish。
- import 只支持 `create_new`，不支持覆盖式 restore。
- project package 只支持本地目录包，不做云同步。
- archive / backup 只做手动本地快照，不做自动定时。
- `/studio` 工作台仍为同步工作台，不提供后台任务队列或复杂运维后台。
- 错误码收口只覆盖成品化 / 工作台直连端点，不做全项目错误协议重构。
