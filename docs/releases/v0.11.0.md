# Release Notes: v0.11.0

## Summary

- 收拢并稳定了 `scene -> chapter -> volume -> book` 的完整最小主链，形成可实际演示与交付的 preview 基线。
- 完成 productization 收尾，覆盖 `export / rebuild / diagnostics / health`、`project package / import-package / archive / backup / snapshots`。
- 增加发布前硬化资产：可直接导入的 sample package、demo workflow、release checklist、release notes 模板、只读 `release_preflight` 检查脚本。
- `/studio` dashboard 已可作为最小工作台使用，覆盖 health、rebuild、project export、import、archive、backup 与 snapshots。

## Demo Workflow

推荐演示路径与 `examples/sample-project-package/` 强绑定：

1. 启动服务：`python -m uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000`
2. 导入 sample package：`POST /api/projects/import-package`
3. 打开 `/studio`
4. 在 dashboard 加载导入项目的 health
5. 触发：
   - `Export Project Package`
   - `Create Archive Snapshot`
   - `Create Backup`
   - `Load Snapshots`

详细步骤与每一步预期结果见：

- `docs/demo-workflow.md`

## Known Limitations

- 当前版本仍是 preview，不是最终 GA 发布。
- 不新增新的智能写作模块。
- 不做 LLM critic。
- 不做 book-level repair / rewrite / polish。
- import 只支持 `create_new`，不支持覆盖式 restore。
- project package 只支持本地目录包，不做云同步。
- archive / backup 只做手动本地快照，不做自动定时。
- `/studio` dashboard 仍为同步工作台，不提供后台任务队列或复杂运维后台。
- 错误码收口只覆盖 productization / dashboard 直连端点，不做全项目错误协议重构。
